# Web App Security

- Authentication: তুমি কে valid user কি না?
- Authorization: কোনো user এর অনুমতি আছে কি না, কিছু একটা ব্যবহার করার জন্য।

- Authentication methods:

1. Session-based
2. Token-based
3. OAuth
